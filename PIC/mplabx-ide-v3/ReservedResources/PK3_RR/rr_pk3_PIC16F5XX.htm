PICkit 3 Reserved Resources By Device

PIC10F2XX, PIC12F/16F5XX Devices

These devices, and any related debug headers, may require general device resources, as well as specific program and data memory resources, to perform debug. Please see the sections below for details. For more information on debug headers, see the "Processor Extention Pak (PEP) and Debug Header Specification" (DS50001292), the "PEP and Debug Header Help" file or the PEP and Debug Header wiki.For more information on emulation headers, see the "Emulation Extention Pak (EEP) and Emulation Header User's Guide" (DS50002243), the "EEP and Emulation Header Help" file or the EEP and Emulation Header wiki.

General Resources Used - Headers

  • MCLR/Vpp shared for programming.

Program and Data Memory Used

In MPLAB X IDE: Program memory and data memory (file register) windows will show an “R” to denote each nibble of data that represents a reserved resource.

In the Device Table: Program memory and data memory (file register) reserved resources are listed by device.
