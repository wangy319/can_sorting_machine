Process

The process is the fundamental code element in SCL. It represents a sequence of events to be performed.

Here is the basic syntax of a process. 

process is
   begin
      // process statements go here
   end process;

A testbench may have more than one process in it. 

testbench for PIC32MX360F512L is
   process is // 1st process
      begin
      end process;
   process is // 2nd process
      begin
      end process;
   end testbench;

Finally, a process may have a name 

processName: process is
   begin
end process processName;

or, alternatively 

processName: is process is
   begin
end process processName;

The process name is decorative only but must be unique. The process name may be used in future versions of SCL.
