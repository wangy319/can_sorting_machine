Arithmetic

Mathmatical operations can be performed on word sized integer values like SFRs and integer variables. 

intVal <= intVal [+|-|*|/] intVal;

Integer values can be SFRs, integer variables, integer constants, or word sized shared variable . 

TMR1 <= TMR1 + 1;
intVar := WREG / 5;
sharedVar <= intVar + RCREG * 3 - 1;
